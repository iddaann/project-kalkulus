Optimasi Biaya Langganan Layanan Streaming

Analisis biaya bulanan berdasarkan penggunaan penyimpanan dengan fungsi piecewise

Hasil Analisis

Analisis Matematis

Fungsi Biaya C(x)

Fungsi piecewise untuk biaya bulanan:

  • Basic: C(x) = 50.000 + 2.000·max(0, x-50) untuk 0 ≤ x ≤ 50
  • Standard: C(x) = 90.000 + 2.000·max(0, x-200) untuk 50 < x ≤ 200
  • Premium: C(x) = 150.000 + 2.000·max(0, x-500) untuk x > 200

Analisis Kekontinuan

Pada x = 50 GB:

  • Limit kiri: limx→50⁻ C(x) = 50.000
  • Limit kanan: limx→50⁺ C(x) = 90.000
  • Kedua limit tidak sama, sehingga fungsi tidak kontinu di x = 50

Pada x = 200 GB:

  • Limit kiri: limx→200⁻ C(x) = 90.000
  • Limit kanan: limx→200⁺ C(x) = 150.000
  • Kedua limit tidak sama, sehingga fungsi tidak kontinu di x = 200

Implikasi bagi Customer

Ketidakkontinuan pada x=50 dan x=200 berarti terjadi lompatan biaya yang signifikan ketika melewati batas paket. Customer perlu berhati-hati dalam memperkirakan penggunaan penyimpanan agar tidak mengalami kenaikan biaya yang tiba-tiba.
